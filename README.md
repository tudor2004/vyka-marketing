## Regenerate

Scripts need Pillow, NumPy, imageio-ffmpeg, and httpx (product photos are fetched
live from store CDNs at render time). Run any generator from the repo root:

```
uv run --with pillow --with numpy --with imageio-ffmpeg --with httpx python scripts/<name>.py
```

Typical order for a campaign: `make_*` elements first, then the matching `build_*`
reel. The reel builders read intermediates from `out/work/` and write the finished
reel to `out/`.
